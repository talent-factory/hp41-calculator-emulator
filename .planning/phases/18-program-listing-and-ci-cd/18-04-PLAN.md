| T-18-W2-02 | Information Disclosure | program_steps rendered in DOM | accept | Program data belongs to the user; no secrets or PII in program listings; rendered in a local Tauri window, not transmitted |
| T-18-W2-03 | Elevation of Privilege | SVG key id: 'sst'/'bst' | accept | KEY_DEFS id values are compile-time constants in a local desktop app; no external injection surface |
</threat_model>
