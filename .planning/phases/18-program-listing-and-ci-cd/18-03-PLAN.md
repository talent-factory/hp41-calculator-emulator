Create `.github/workflows/ci-gui.yml` with the following exact content. Do NOT modify `.github/workflows/ci.yml` — that file must remain untouched (SC-5 independence invariant).

```yaml
name: ci-gui

on:
  push:
    branches: [main, develop]
    paths:
      - 'hp41-gui/**'
      - 'hp41-core/**'
  pull_request:
    branches: [main, develop]
    paths:
      - 'hp41-gui/**'
      - 'hp41-core/**'

env:
  CARGO_TERM_COLOR: always
  RUST_BACKTRACE: 1

jobs:
  build:
    name: GUI Build (${{ matrix.os }})
    runs-on: ${{ matrix.os }}
    strategy:
      fail-fast: false
      matrix:
        os: [ubuntu-latest, macos-latest, windows-latest]

    steps:
      - uses: actions/checkout@v4

      - uses: dtolnay/rust-toolchain@stable

      - uses: Swatinem/rust-cache@v2
        with:
          workspaces: hp41-gui/src-tauri -> hp41-gui/src-tauri/target

      - uses: actions/setup-node@v4
        with:
          node-version: 'lts/*'

      - name: Install Linux system deps
        if: matrix.os == 'ubuntu-latest'
        run: sudo apt-get update && sudo apt-get install -y libwebkit2gtk-4.1-dev libappindicator3-dev librsvg2-dev patchelf

      - name: Install npm dependencies
        run: npm install
        working-directory: hp41-gui

      - name: TypeScript type check
        run: npx tsc --noEmit
        working-directory: hp41-gui

      - name: Rust build (release)
        run: cargo build --release
        working-directory: hp41-gui/src-tauri
```

Notes on the Swatinem/rust-cache@v2 `workspaces` key: the `workspaces` parameter tells the cache where the Cargo manifest is relative to the repo root. Use `hp41-gui/src-tauri -> hp41-gui/src-tauri/target` to cache the GUI crate's target directory separately from any root workspace cache. This matches the nested workspace isolation established in Phase 13.

The `working-directory: hp41-gui` for `npm install` and `npx tsc --noEmit` is correct because `package.json` and `tsconfig.json` live in `hp41-gui/` (not `hp41-gui/src-tauri/`).

The `working-directory: hp41-gui/src-tauri` for `cargo build --release` is correct because `hp41-gui/src-tauri/Cargo.toml` is the Rust manifest for the GUI crate.
