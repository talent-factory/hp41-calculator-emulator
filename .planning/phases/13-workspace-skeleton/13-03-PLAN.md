**Step 2: Create hp41-gui/src-tauri/capabilities/default.json**

Tauri v2 requires at least one capability file to be present. Without it, `tauri dev` fails
with "no capabilities found" or similar.

Exact content (standard Tauri v2 default capability):

```json
{
  "identifier": "default",
  "description": "Default capability for hp41-gui",
  "windows": ["main"],
  "permissions": [
    "core:default"
  ]
}
```

Create the directory first: `mkdir -p hp41-gui/src-tauri/capabilities`

Key constraint: `"core:default"` grants standard Tauri core permissions (window management,
IPC). This is the minimum required for a Tauri app to function. Additional permissions
(e.g., file system access for persistence) are added in Phase 17.
