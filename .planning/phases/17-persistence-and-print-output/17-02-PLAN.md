<objective>
Add a collapsible, auto-showing print panel to the GUI React frontend that accumulates print output from PRX/PRA/PRSTK and displays it in a scrollable dark-themed panel consistent with the calculator aesthetic.

Purpose: Print output from hp41-core's `print_buffer` is currently drained by the IPC layer into `CalcStateView.print_lines` on every call — but the frontend silently discards it. This plan makes that output visible to the user (PERS-02, SC-5).

Output: `App.tsx` extended with 3 state declarations + 2 useEffects + print panel JSX; `App.css` extended with print panel CSS rules (~30 lines).
</objective>
