## Remaining Tech Debt: 2 minor items

### Area 1: EEX trailing-e behavior (Phase 08 residual)
- Typing `1.5e` (mantissa without exponent digits) then pressing a non-digit discards the number silently
- Documented with a test (`test_flush_trailing_e_without_exponent_returns_err`)
- Non-blocking: no panic, error path is safe; HP-41 hardware prevents this via UI lock

### Area 2: STO arithmetic keyboard modals (deliberately deferred)
- `PendingInput::StoAdd/Sub/Mul/Div` remain dead code at keyboard level
- STO arithmetic accessible via recorded programs as designed
- Deferred to v1.1: requires multi-step modal flow design

**Total: 2 minor items — neither blocks release.**
